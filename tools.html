  • pomfload

    POSIX-compatible shell script

    Upload files directly from the command line. Just change the second line of the script to read uppomf="https://smutty.horse/upload.php"
    • Download
    • Contact

  • limf

    Python-based uploading and encrypting

    Can upload and encrypt/decrypt files to any pomf clones (including Smutty Horse). Also includes a tool for making screenshots automatically
    Download

  • Pomf Rehost

    Chrome browser extension

    This should work if you choose Custom and use https://smutty.horse as the host name. Not yet tested

  • Gyazowin

    This appears to be a modified form of Gyazo. You'll want to compile this from source to use it with Smutty Horse. Change line 794 in gyazowin/gyazowin.cpp to read: 
    const TCHAR* UPLOAD_SERVER = _T("smutty.horse");
    No guarantees this will work
    • Download
    • Contact

  • Pomfshare

    Android share client

    Adds a Pomfshare destination to your sharing menu on Android. To get this to work with Smutty Horse, you'll need to modify at least the app/src/main/java/science/itaintrocket/pomfshare/HostListActivity.java file and add an entry that would look something like: 
    hosts[24] = new Host("smutty.horse", "https://smutty.horse/upload.php?output=gyazo", "128MiB", Host.Type.POMF);
    No guarantees this one will work, though I am curious and may try to roll a version specifically for Smutty Horse at some point
