Smutty Horse Tools

Easier uploading to Smutty Horse!

  • Progressive Web App

    Enables Android and Chrome sharing

    The Smutty.Horse Progressive Web App includes the ability to act as the target of a share request. On Android, this means it is added to the list of sharing destinations. Every 30 days, you should be prompted to install the app if you haven't already. Alternatively, on the home page of the site the text install Smutty Horse should be a link if your operating system is capable of installing PWAs.

  • ShareX

    Windows-based screen capture and file sharing

    Easy to use and very powerful, this open source program is the standard tool for Windows users. After installation, be sure to use our custom uploader

  • pomfload

    POSIX-compatible shell script

    Upload files directly from the command line. Just change the second line of the script to read uppomf="https://smutty.horse/upload.php"

  • limf

    Python-based uploading and encrypting

    Can upload and encrypt/decrypt files to any pomf clones (including Smutty Horse). Also includes a tool for making screenshots automatically
